Collaborative Music Neurofeedback

A March 28, 2026 build log and proposal for live EEG, Spotify-linked track switching, Cloudflare publication, and remote collaborator feedback.

Open Markdown Proposal Open Live Dashboard Open Spectrum Page

What Was Built

  • Real-time OpenBCI Cyton EEG analyzer in Python.
  • Cloudflare Worker plus D1 archival for public live and historical views.
  • Spotify track-change monitor with EEG before/after screenshots.
  • Webcam-based eye state and coarse event tagging.
  • Experiment annotations for nicotine and sleep.

Why It Matters

The system is already good enough for remote observers to watch live brainwave metrics, inspect archived PSD snapshots, and review how track switches changed artifact burden and spectral shape.

The next phase is collaborative control: let remote people suggest tracks and rank them by measured neural response.
